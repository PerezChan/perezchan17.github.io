Web Scraping all Locations of Elderly Homes and Plotting onto a Map

For a data visualisation project I wanted to plot the locations of all Singaporean Elderly Homes. Doing it manually would take far too much time so I managed to find a local website with all the elderly homes and scraped them.


I then inputted the zipcode of every home into a longitude and latitude website and appended it into a csv file for my data visualisation in Tableau.
