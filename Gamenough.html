Project Requirement

Create a Gamified Finance Project to Educate the Youths

1

Main Idea and Inspiration

Growing up with a Nintendo DS, I played Pokemon Emerald and Firered excessively. For this project I decided to put in the extra effort and make a game prototype to gamify the learning process.


Pokemon Emerald is a 2D pixel game with character interactions and story-lines.

2

Choosing Gamemaker to make my Prototype Game

Gamemaker has their own proprietary scripting language GML. This was quite a big milestone for me because I have never learnt a new language fully by myself, the main reason I chose Gamemaker over other game softwares is that Gamemaker has many tutorial on Youtube and the interface is similar to VS Code which I am very familiar with.

3

Using Free-to-use Assets to bring my Game to life

I scowered the internet to find the most suitable pixel assets to resemble Pokemon as much as possible. I wanted users to have a sense of familiarity when they play my game and intuitively know how to play the game.


This project took a long time but was incredibly rewarding. I even put in the extra effort to add walking effects and shadows on my game character and also designed some of the assets to find the setting even better myself.
